2. Data yang kami kumpulkan — dan yang tidak kami kumpulkan

2.1 Situs web ini

Situs web editoria11y.com adalah situs statis yang dihosting di GitHub Pages. Situs ini tidak menetapkan cookie, tidak menjalankan skrip analitik, dan tidak mengumpulkan atau menyimpan data pribadi apa pun secara mandiri. Tidak ada piksel pelacakan, sidik jari digital, atau pembuatan profil perilaku dalam bentuk apa pun yang terjadi di halaman kami.

2.2 Log hosting

GitHub Pages mungkin mengumpulkan log akses server yang terbatas (seperti alamat IP) sebagai bagian dari infrastrukturnya. Pemrosesan ini diatur oleh Pernyataan Privasi GitHub. Kami tidak memiliki akses ke log tersebut.

3. Layanan pihak ketiga yang mungkin memproses data Anda

Meskipun situs web itu sendiri tidak mengumpulkan data pribadi, layanan tertaut tertentu yang kami andalkan untuk pembelian, komunikasi, dan manajemen proyek memang memproses data pribadi. Ketika Anda berinteraksi dengan layanan-layanan ini, kebijakan privasi mereka masing-masing berlaku.

3.1 Freemius — pembelian dan lisensi

Jika Anda membeli langganan add-on Editoria11y, transaksi tersebut diproses oleh Freemius, Inc. Freemius mengumpulkan detail pembayaran, nama, alamat email, dan informasi penagihan terkait yang diperlukan untuk menyelesaikan dan mengelola pembelian Anda. Freemius bertindak sebagai pengontrol data (untuk operasi layanannya sendiri) dan pemroses data (atas nama kami untuk manajemen lisensi).

  • Kebijakan Privasi Freemius
  • Perjanjian Pemrosesan Data Freemius

Dasar hukum: Pelaksanaan kontrak (Pasal 6(1)(b) GDPR) dan kepentingan sah kami dalam mengoperasikan bisnis (Pasal 6(1)(f)).

3.2 JotForm — formulir kontak

Formulir kontak kami disediakan oleh JotForm Inc. Ketika Anda mengirimkan formulir, JotForm mengumpulkan informasi yang Anda berikan (biasanya nama, alamat email, dan pesan Anda). Data pengiriman dihosting di server berbasis UE milik JotForm.

  • Kebijakan Privasi JotForm
  • Addendum Pemrosesan Data JotForm

Dasar hukum: Persetujuan Anda ketika Anda memilih untuk mengirimkan formulir (Pasal 6(1)(a)) dan kepentingan sah kami dalam menanggapi pertanyaan (Pasal 6(1)(f)).

3.3 Google Workspace — email

Jika Anda mengirim email kepada kami, pesan Anda diproses melalui Google Workspace (Gmail). Google dapat memproses alamat pengirim dan penerima, konten pesan, dan metadata sesuai dengan komitmen perlindungan datanya sendiri.

  • Kebijakan Privasi Google
  • Addendum Pemrosesan Data Google Workspace

Dasar hukum: Kepentingan sah kami dalam berkomunikasi dengan pengguna dan mitra (Pasal 6(1)(f)).

3.4 GitHub & GitLab — antrean isu dan forum

Laporan bug, permintaan fitur, dan diskusi komunitas berlangsung di GitHub dan GitLab. Ketika Anda membuat akun atau memposting di platform ini, mereka memproses nama pengguna, alamat email, dan konten yang Anda kirimkan.

  • Pernyataan Privasi GitHub
  • Kebijakan Privasi GitLab

Dasar hukum: Kepentingan sah kami dalam memelihara dan meningkatkan proyek (Pasal 6(1)(f)).

4. Transfer data internasional

Beberapa layanan pihak ketiga yang tercantum di atas berbasis di Amerika Serikat. Jika data pribadi ditransfer di luar Kawasan Ekonomi Eropa (KEE), penyedia ini mengandalkan perlindungan yang diakui berdasarkan GDPR, termasuk Kerangka Privasi Data UE-AS, Klausul Kontraktual Standar (SCC), atau mekanisme transfer yang disetujui lainnya. Silakan merujuk ke dokumentasi privasi masing-masing penyedia untuk detailnya.

JotForm menyimpan data pengiriman formulir kontak di server berbasis UE, meminimalkan transfer lintas batas untuk data tersebut.

5. Retensi data

Situs web ini tidak mengumpulkan data pribadi secara langsung.

Data yang dipegang oleh layanan pihak ketiga disimpan sesuai dengan kebijakan mereka masing-masing dan kewajiban hukum yang berlaku. Kami secara berkala meninjau dan menghapus data yang tidak lagi diperlukan.

6. Hak Anda berdasarkan GDPR

Jika Anda berada di KEE, Inggris, atau Swiss, Anda memiliki hak-hak berikut terkait data pribadi Anda:

  • Hak akses — minta salinan data pribadi yang kami simpan tentang Anda.
  • Hak untuk rektifikasi — minta kami untuk memperbaiki data yang tidak akurat.
  • Hak untuk dihapus — minta kami untuk menghapus data pribadi Anda, tunduk pada kewajiban retensi hukum.
  • Hak untuk membatasi pemrosesan — minta kami untuk membatasi cara kami menggunakan data Anda.
  • Hak atas portabilitas data — terima data Anda dalam format terstruktur yang dapat dibaca mesin.
  • Hak untuk keberatan — keberatan terhadap pemrosesan berdasarkan kepentingan sah kami.
  • Hak untuk menarik persetujuan — jika pemrosesan didasarkan pada persetujuan, Anda dapat menariknya kapan saja tanpa memengaruhi keabsahan pemrosesan sebelumnya.

Untuk menggunakan hak-hak ini, hubungi kami di privacy@editoria11y.com. Kami akan merespons dalam 30 hari.

Anda juga memiliki hak untuk mengajukan pengaduan kepada otoritas pengawas. Daftar otoritas perlindungan data KEE tersedia di edpb.europa.eu.
