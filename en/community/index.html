Membership

Editoria11y project members guide & support the development of the Editoria11y library and its CMS plugins.

Members also have access to the "Community Supported Add-ons" (CSA) suite for Drupal and WordPress (eta mid-2026):

CSA plugins

  • Add developer and contrast tests to the core test suite.
  • Create "split configuration" for separate rulesets for developers and content creators.
  • Allow site-wide issue dismissals
  • Provide dashboard maintenance tools and crawlers
  • Gain features from the project roadmap

CSA Community

  • Given priority support and over-weighted voting in issue queues and on project Slack channels.
  • Can test and influence in-development features.
  • (Optional) can ask for public supporter credit in this page.
  • Receives direct assistance* with plugin installation and configuration.

* Everyone is welcome to ask for help on Slack and the issue queues. Those contributing more than $200/year may request to direct some of their support towards consulting/advising hours rather than project development.

Join

Membership is tracked via subscriptions to CSA plugins, but not all members contribute financially. Pathways to membership include:

  • Monthly/annual per-site subscriptions.
  • Unlimited-site enterprise subscriptions.
  • Subscription credits from directly sponsored work.
  • Subscription credits for approved contributions of time and effort.
  • Subscription credits for pre-approved major contributions of time and effort to improve the accessibility of related open source projects.

Subscriptions list at $10/site/month as of 2026, with various discounts listed on the plugin licensing pages. Expect the list price to scale with inflation. If future features require cloud APIs we will talk as a community whether to bundle those hosting costs or offer them as an optional add-on.

Project maintainers consider granting contribution credits based on effort and impact. Please seek pre-approval if you are working under the assumption of approval or plan to request for more than a one-site 12-month credit.
