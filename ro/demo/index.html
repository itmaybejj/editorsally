Text subliniat

Puțin italic este bine, dar blocuri mari de text accentuat nu sunt foarte lizibile. Puțin este acceptabil, desigur, dar există limite, iar un paragraf întreg este mult prea mult. Acesta, de exemplu, este un paragraf întreg. Este oarecum greu de citit, nu? Dar acest test nu se declanșează până nu este atât de lung, deci nu ar trebui să vedeți fals pozitive în conținutul dvs. dacă nu scrieți paragrafe cu adevărat mari ca acesta în mod regulat.
