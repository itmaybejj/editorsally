2. 我们收集哪些数据——以及我们不收集哪些数据

2.1 本网站

editoria11y.com 网站是一个托管于 GitHub Pages 的静态网站。它不设置 cookie，不运行分析脚本，也不自行收集或存储任何个人数据。我们的页面不使用任何类型的跟踪像素、指纹识别或行为画像。

2.2 托管日志

作为其基础设施的一部分，GitHub Pages 可能会收集有限的服务器访问日志（例如 IP 地址）。该处理受 GitHub 隐私声明约束。 我们无法访问这些日志。

3. 可能处理您数据的第三方服务

虽然网站本身不收集个人数据，但我们在购买、通信和项目管理方面所依赖的某些链接服务确实会处理个人数据。当您与这些服务交互时，适用其各自的隐私政策。

3.1 Freemius——购买与许可

如果您购买 Editoria11y 附加订阅，交易将由 Freemius, Inc. 处理。Freemius 会收集完成和管理您购买所必需的付款详情、姓名、电子邮件地址和相关账单信息。Freemius 同时作为数据控制者（针对其自身的服务运营）和数据处理者（代表我们进行许可管理）。

  • Freemius 隐私政策
  • Freemius 数据处理协议

法律依据：履行合同（GDPR 第 6(1)(b) 条）以及我们运营业务的合法利益（第 6(1)(f) 条）。

3.2 JotForm——联系表单

我们的联系表单由 JotForm Inc. 提供。当您提交表单时，JotForm 会收集您提供的任何信息（通常包括您的姓名、电子邮件地址和消息）。提交的数据托管在 JotForm 位于欧盟的服务器上。

  • JotForm 隐私政策
  • JotForm GDPR 合规

法律依据：您选择提交表单时表示的同意（第 6(1)(a) 条）以及我们回复询问的合法利益（第 6(1)(f) 条）。

  • Google 隐私政策
  • Google Workspace 数据处理修正案

3.4 GitHub 与 GitLab——问题队列和论坛

4. 国际数据传输

上述某些第三方服务位于美国。当个人数据传输至欧洲经济区（EEA）之外时，这些提供商依赖 GDPR 认可的保障措施，包括欧盟-美国数据隐私框架、标准合同条款（SCCs）或其他经批准的传输机制。详情请参阅各提供商的隐私文档。

JotForm 将联系表单提交的数据存储在位于欧盟的服务器上，从而最大限度地减少此类数据的跨境传输。

5. 数据保留

由于本网站不直接收集个人数据，我们不维护用户数据库。第三方服务持有的数据根据其各自的政策和任何适用的法律义务保留。我们会定期审查并删除不再需要的数据。

6. 您在 GDPR 下的权利

如果您位于欧洲经济区、英国或瑞士，您对自己的个人数据享有以下权利：

  • 访问权——请求获取我们持有的关于您的任何个人数据的副本。
  • 更正权——要求我们更正不准确的数据。
  • 删除权——要求我们删除您的个人数据，但需遵守法定保留义务。
  • 限制处理权——要求我们限制对您数据的使用。
  • 数据可携权——以结构化、机器可读的格式接收您的数据。
  • 反对权——反对基于我们合法利益的处理。
  • 撤回同意权——在处理基于同意的情况下，您可以随时撤回同意，且不影响先前处理的合法性。
