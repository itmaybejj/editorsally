Texto sublinhado

Um pouco de itálico é aceitável, mas grandes blocos de texto enfatizado não são muito legíveis. Um pouco é tolerável, claro, mas há limites, e um parágrafo inteiro já é demais. Este, por exemplo, é um parágrafo inteiro. É um pouco difícil de ler, não é? Mas este teste só é acionado quando o texto é tão longo assim, portanto você não deve ver falsos positivos no seu conteúdo, a menos que escreva parágrafos muito longos regularmente como este.

Texto justificado

Uso excessivo de sobrescrito ou subscrito; o subscrito deve ser usado para referências pequenas e notas de rodapé, não para frases ou parágrafos inteiros.
